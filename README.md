# Magic: The Distributed Cognition Workshop

Welcome to the workshop of the Mages' Alliance.

## Architecture

This repository (`magic`) contains:

*   **The Workshop**—active desk, tools, and system configuration (`system/`, `desk/`, `floor/`, `box/`)
*   **The Library**—Alliance wisdom scrolls for Spirit consultation (`library/`)
*   **The Observatory**—public research infrastructure (`library/observatory/`)

The Library is a directory of scrolls tracked in this repository. Spirits consult it for Alliance wisdom, protocols, and guidance.
